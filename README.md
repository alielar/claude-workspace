# Claude Workspace — Master Folder

This is the master folder for all Claude + Cursor projects. Every new project gets its own subfolder under `projects/`.

---

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `projects/` | **One subfolder per project** — create a new one here each time you start something |
| `skills/` | Custom Claude Code skills & slash commands you've built |
| `templates/` | Reusable templates (prompts, project skeletons, scripts) |
| `code/` | Standalone scripts and code not tied to a specific project |
| `notes/` | General notes, summaries, and documentation |
| `research/` | Research, analysis, and references |
| `data/` | Datasets, exports, and shared data files |

---

## How to Start a New Project

1. Create a new folder inside `projects/` — name it clearly (e.g., `2026-04-eduease-chatbot`)
2. Inside the project folder, add whatever you need: code, notes, data, etc.
3. Optionally add a `README.md` inside the project folder to document what it does

---

## Skills

Custom Claude Code skills and slash commands live in `skills/`.
To use a skill globally in Claude Code, symlink or copy it to `~/.claude/commands/`.
