# Skills

Custom Claude Code slash commands. These are used in the terminal via `claude` or Cursor.

| Skill | What it does |
|-------|-------------|
| `explore.md` | Initial exploration stage — understand the codebase before touching anything |
| `create-plan.md` | Produce a structured markdown plan based on the full conversation |
| `execute.md` | Execute the implementation plan precisely |
| `review.md` | Comprehensive code review |
| `peer-review.md` | Analyze findings from a peer reviewer with less project context |
| `document.md` | Update documentation after code changes |
| `create-issue.md` | Quickly capture a bug/feature idea mid-development |
| `learning-opportunity.md` | Pause dev mode and get a technical explanation of what's happening |

## To use these in Claude Code

These files are already active at `~/.claude/skills/`. This folder is a backup/reference copy.
